LINUX:
1. Right click on your project folder and choose Properties (Alt+Enter)
2. Go to C/C++ Build -> Settings and in GCC Compiler/Includes add the following path:
	/usr/include/glib-2.0
	/usr/lib/x86_64-linux-gnu/glib-2.0/include
	/usr/local/include
	/usr/local/include/pbc
3. Go to C/C++ Build -> Settings and in GCC Compiler/Optimization select "Optimize most (-O3)" as Optimization Level
4. Go to C/C++ Build -> Settings and in GCC Compiler/Debugging select "None" as Debug Level
5. Go to C/C++ Build -> Settings and in GCC Compiler/Warnings tick "All warnings (-Wall)" checkbox
6. Go to C/C++ Build -> Settings and in GCC Compiler/Miscellaneous set the following options on "Other flags":
	-c -DPACKAGE_NAME=\"cpabe\" -DPACKAGE_TARNAME=\"cpabe\" -DPACKAGE_VERSION=\"0.11\" -DPACKAGE_STRING=\"cpabe\ 0.11\" -DPACKAGE_BUGREPORT=\"bethenco@cs.berkeley.edu\" -DPACKAGE_URL=\"\" -DSTDC_HEADERS=1 -DHAVE_SYS_TYPES_H=1 -DHAVE_SYS_STAT_H=1 -DHAVE_STDLIB_H=1 -DHAVE_STRING_H=1 -DHAVE_MEMORY_H=1 -DHAVE_STRINGS_H=1 -DHAVE_INTTYPES_H=1 -DHAVE_STDINT_H=1 -DHAVE_UNISTD_H=1 -DSTDC_HEADERS=1 -DHAVE_FCNTL_H=1 -DHAVE_STDDEF_H=1 -DHAVE_STRING_H=1 -DHAVE_STDLIB_H=1 -DHAVE_MALLOC=1 -DLSTAT_FOLLOWS_SLASHED_SYMLINK=1 -DHAVE_VPRINTF=1 -DHAVE_LIBCRYPTO=1 -DHAVE_LIBCRYPTO=1 -DHAVE_STRCHR=1 -DHAVE_STRDUP=1 -DHAVE_MEMSET=1 -DHAVE_GMP=1 -DHAVE_PBC=1 -DHAVE_BSWABE=1
7. Go to C/C++ Build -> Settings and in GCC Linker add the following options at the end of "Command line pattern":
	-O3 -Wall -lglib-2.0 -Wl,-rpath /usr/local/lib -lgmp -Wl,-rpath /usr/local/lib -lpbc -lbswabe -lcrypto -lcrypto -lgmp

WINDOWS:
1. Right click on your project folder and choose Properties (Alt+Enter)
2. Go to C/C++ Build -> Settings and in GCC Compiler/Includes add the following path (may depend on configuration in installiation phase):
	"C:\cygwin64\usr\include\glib-2.0"
	"C:\cygwin64\lib\glib-2.0\include"
	"C:\cygwin64\usr\local\include\pbc"
3. Go to C/C++ Build -> Settings and in Cygwin Compiler/Optimization select "Optimize most (-O3)" as Optimization Level
4. Go to C/C++ Build -> Settings and in Cygwin Compiler/Debugging select "None" as Debug Level
5. Go to C/C++ Build -> Settings and in Cygwin Compiler/Warnings tick "All warnings (-Wall)" checkbox
6. Go to C/C++ Build -> Settings and in Cygwin Compiler/Miscellaneous set the following options on "Other flags":
	-c -DPACKAGE_NAME=\"cpabe\" -DPACKAGE_TARNAME=\"cpabe\" -DPACKAGE_VERSION=\"0.11\" -DPACKAGE_STRING=\"cpabe\ 0.11\" -DPACKAGE_BUGREPORT=\"bethenco@cs.berkeley.edu\" -DPACKAGE_URL=\"\" -DSTDC_HEADERS=1 -DHAVE_SYS_TYPES_H=1 -DHAVE_SYS_STAT_H=1 -DHAVE_STDLIB_H=1 -DHAVE_STRING_H=1 -DHAVE_MEMORY_H=1 -DHAVE_STRINGS_H=1 -DHAVE_INTTYPES_H=1 -DHAVE_STDINT_H=1 -DHAVE_UNISTD_H=1 -DSTDC_HEADERS=1 -DHAVE_FCNTL_H=1 -DHAVE_STDDEF_H=1 -DHAVE_STRING_H=1 -DHAVE_STDLIB_H=1 -DHAVE_MALLOC=1 -DLSTAT_FOLLOWS_SLASHED_SYMLINK=1 -DHAVE_VPRINTF=1 -DHAVE_LIBCRYPTO=1 -DHAVE_LIBCRYPTO=1 -DHAVE_STRCHR=1 -DHAVE_STRDUP=1 -DHAVE_MEMSET=1 -DHAVE_GMP=1 -DHAVE_PBC=1 -DHAVE_BSWABE=1
7. Go to C/C++ Build -> Settings and in Cygwin Linker add the following options at the end of "Command line pattern":
	-O3 -Wall -lglib-2.0 -lintl -Wl,-rpath "C:\cygwin64\usr\local\lib" -lgmp -L/usr/local/lib/ -Wl,-rpath "C:\cygwin64\usr\local\lib" -Wl,-rpath "C:\cygwin64\usr\local\lib"  -lpbc -lbswabe -lcrypto -lcrypto -lglib-2.0 -lgmp -lpbc
